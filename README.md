## 🌱分支说明

> 该项目主要存在三个分支，第一个分支为基础模块（`base-dev`），第二个分支是集成 Security 模块（`security-dev`），第三个为主分支（`master`）

| 分支名称         | 分支说明                                     | 分支地址 |
| :--------------- | -------------------------------------------- | :------: |
| master           | 主分支，目前开发主要在 `master` 分支上面进行 |          |
| base-develop     | 基础模块分支，包含了系统的基础模块和前端模块 |          |
| security-develop | 认证、授权模块分支，基于 `base-develop` 分支 |          |
